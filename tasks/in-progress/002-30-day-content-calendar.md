## Production Workflow

### Per Article Process

1. **Keyword Research** (Day before publishing)
   - Check volume/difficulty with DataForSEO
   - Analyze SERP for format/angle
   - Review top 3 competitors
   - Document target keywords (primary + secondary)

2. **Brief Creation** (Morning of writing day)
   - Objective (what this article achieves)
   - Target audience (who we're writing for)
   - Angle/unique value prop
   - Outline structure (H2s, H3s)
   - SEO notes (keywords, internal links, external links)

3. **Writing** (2-3 hours per article)
   - Word count: 800-1,500 (informational), 1,500-2,500 (commercial)
   - Include: Stats, examples, actionable advice
   - Structure: Short paragraphs, bullets, tables
   - Tone: Informative, helpful, not overly salesy

4. **SEO Optimization** (30 min after writing)
   - Primary keyword in title, H1, URL slug, first 100 words
   - Secondary keywords in H2s and body
   - Meta description (150-160 chars)
   - 2-5 internal links to relevant pages
   - 1-3 external links to authoritative sources
   - Image alt text (if using images)

5. **Review & Publish** (Final check)
   - Read against SEO checklist (see SOUL.md)
   - Check for thin content or duplicate sections
   - Verify CTA is clear (affiliate link placement)
   - Publish via WordPress REST API
   - Submit URL to Google for indexing
