## Cross-Linking Strategy

**Decision:** Should sites link to each other?

**Pros:**
- Boosts all sites with internal links
- Shares traffic between properties
- Easier to rank network as a whole

**Cons:**
- Google may detect PBN (private blog network)
- If one site penalized, all affected
- Harder to sell individual sites later

**Recommendation:**
- **Limited cross-linking only** - when genuinely relevant
- Example: freecrashgames.com "Best Crash Casinos for China" can link to cryptocrashgambling.com "Bitcoin Crash Gambling" if relevant
- Don't force it - topical relevance must be clear
- Use natural anchor text (not exact-match keywords)
- No site-wide footer links
