## What Is a Crash Hash?

A **crash hash** is a cryptographic fingerprint used by crash gambling games to prove fairness. It's the mathematical foundation that ensures crash multipliers are random and can't be manipulated.

### The Hash Function

Most crash casinos use **SHA-256** (Secure Hash Algorithm 256-bit):

```
Input: Server Seed + Client Seed + Nonce
Output: Hash (crash multiplier)
```

**Key Properties:**
- **Deterministic:** Same input always produces same output
- **One-way:** Can't reverse hash to find input
- **Unique:** Even tiny input changes completely different hash
- **Verifiable:** Anyone can confirm the math

## How Crash Games Generate Multipliers

### Step 1: Seed Generation

**Server Seed (Casino):**
- Secret random number
- Generated by casino
- Hashed and shown BEFORE game starts

**Client Seed (Player):**
- Random number you provide
- Or generated by browser
- Combined with server seed

**Nonce:**
- Incrementing counter
- Ensures unique result every round

### Step 2: Hash Calculation

```
SHA-256(server_seed + client_seed + nonce) = hash_value
```

### Step 3: Convert to Crash Point

The hash gets converted to a crash multiplier:

```
house_edge = 0.01 (1%)
crash_point = (hash_value % 100) / 100 + house_edge

Example:
hash_value = 123456789
crash_point = (89 / 100) + 0.01 = 0.90x
```

## Why Crash Games Are Mathematically Fair

### The House Edge Explained

Crash games have a **built-in house edge** (usually 1-4%):

```
Average crash multiplier = 0.99x (1% house edge)
Expected return per $1 bet = $0.99
Long-term: Casino profits 1%
```

**Example:**
- You bet $1 at 2.00x multiplier
- 50% chance to win $2
- 50% chance to lose $1
- Expected value = ($2 × 0.5) + (-$1 × 0.5) = $0.50 loss
- **House edge ensures long-term profit for casino**

### Provably Fair vs. House Edge

**Provably fair** = Game isn't rigged, but still has house edge
**House edge** = Mathematical advantage for the casino

Both can exist together:
- Games are provably fair (random, verifiable)
- Casino still profits in long run (house edge)
