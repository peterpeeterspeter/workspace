## Technical Implementation Notes

**Email Service Choice (Awaiting Peter's decision):**

**Option 1: Mailgun (Free Tier)**
- Cost: Free (up to 5,000 emails/month)
- Pros: API-first, programmatic, integrates well
- Cons: Manual list management, basic analytics
- Best for: Technical implementation, API automation

**Option 2: ConvertKit (Paid)**
- Cost: ~$9/month starting
- Pros: Visual automation builder, better analytics, subscriber segmentation
- Cons: More expensive, steeper learning curve
- Best for: Marketing teams, visual workflow management

**Recommendation:** Start with Mailgun free tier. Upgrade to ConvertKit if:
- List exceeds 5,000 subscribers
- Need advanced automation
- Want better analytics
- Willing to pay $9-25/month
