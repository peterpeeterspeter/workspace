## Page Templates

### 1. Program Listing Page Template

**Components:**
- **Hero:** Title + meta description (SEO)
- **Intro:** What is [X type] online MBA?
- **Quick Stats:** Average tuition, duration, salary
- **Program Table:** Sortable/filterable (name, tuition, ranking, format)
- **Top 3-5 Programs:** Detailed cards with affiliate links
- **All Programs:** Grid/list view
- **FAQ:** Common questions
- **CTA:** "Find Your Program" or "Request Info"
- **Affiliate Links:** Prominent, tracked

**Data Points Displayed:**
- School name + logo
- Program name
- Tuition (total)
- Duration (months)
- Format (online/hybrid)
- GMAT required?
- US News / Fortune ranking
- Specializations
- CTA button: "Visit School Site" (affiliate link)
- CTA button: "Request Information" (affiliate link)

### 2. Program Review Page Template

**Components:**
- **Hero:** School name + program name + rating
- **Quick Stats Box:** Tuition, duration, ranking, salary
- **Program Overview:** Description, highlights
- **Curriculum:** Courses, structure, format
- **Admissions:** Requirements, acceptance rate, deadlines
- **Cost & Aid:** Tuition breakdown, scholarships, ROI
- **Pros & Cons:** Honest assessment
- **Student Reviews:** Testimonials (if available)
- **Comparison:** Similar programs
- **CTA:** Multiple affiliate link placements
- **FAQ:** Program-specific questions

**SEO Elements:**
- Target keyword: "[School Name] Online MBA Review"
- LSI keywords: school name + online MBA
- Schema: Review schema (Article + EducationalOrganization)
- Internal links to related programs

### 3. Comparison Page Template

**Components:**
- **Hero:** Comparison title (e.g., "Best Affordable Online MBAs 2026")
- **Intro:** Why this comparison matters
- **Methodology:** How we ranked/compared
- **Comparison Table:** Side-by-side metrics
- **Detailed Breakdown:** Each program section
- **Winner's Podium:** #1, #2, #3 highlights
- **Who Should Choose:** Target audience for each
- **FAQ:** Comparison-specific questions
- **CTA:** Program finder tool

**Data Compared:**
- Tuition
- Duration
- Ranking

## Affiliate Strategy

### Affiliate Program Targets

**1. University Direct Partnerships:**
- Many universities have direct affiliate programs
- Commission: $50-$500 per lead/enrollment
- High intent = good conversion

**2. MBA Aggregators:**
- MBA.com (GMAC)
- GradSchools.com
- StudyPortals
- Commission: 5-15% per referral

**3. Test Prep Providers:**
- GMAT prep courses (Magoosh, Kaplan, Manhattan Prep)
- GRE prep (same providers)
- Commission: 10-20% per sale

**4. Application Consulting:**
- MBA admissions consultants
- Essay editing services
- Commission: 15-25% per sale

**5. Online Learning Platforms:**
- Coursera, edX (certificate programs)
- LinkedIn Learning
- Commission: 5-10% per sale

### Affiliate Link Placement

**High-Value Placements:**
1. **Program cards** in listings (primary CTA)
2. **Review pages** (multiple CTAs: "Visit Site", "Request Info")
3. **Comparison tables** (direct links to schools)
4. **Calculator tools** (result pages with recommendations)
5. **Sidebar widgets** ("Top Programs", "Featured")
6. **Exit-intent popups** (lead capture)

**Link Types:**
- Direct school links (highest intent)
- "Request Information" forms (lead gen)
- Free info guides (email capture)
- Scholarship searches (engagement)

## Technical Requirements

### Platform Choice

**Recommendation:** WordPress + Custom Theme
- **Why:** Affiliate-friendly, easy content management, plugin ecosystem
- **Key Plugins Needed:**
  - Affiliate link management (ThirstyAffiliates or Pretty Links)
  - Table builder (TablePress or wpDataTables)
  - Schema markup (Schema Pro or RankMath)
  - Page builder (Elementor or Gutenberg)
  - Caching (WP Rocket)
  - SEO (RankMath or Yoast)

### Database Structure

**Custom Post Types:**
- `mba_programs` (programs database)
- `mba_reviews` (detailed reviews)
- `mba_comparisons` (comparison pages)

**Custom Taxonomies:**
- `program_type` (online, hybrid, executive)
- `specialization` (finance, marketing, etc.)
- `state` (for location-based filtering)
- `accreditation` (AACSB, ACBSP, etc.)
- `gmat_requirement` (required, optional, waived)

**Custom Fields (ACF or Meta Box):**
- Tuition (total, per credit)
- Duration
- Credits required
- GMAT required (yes/no)
- GRE accepted (yes/no)
- Acceptance rate
- Class size
- Post-grad salary
- Employment rate
- US News ranking
- Fortune ranking
- Specializations (array)
- Start dates
- Application deadline
- Program URL
- Affiliate link
- School logo
- Program description

### Features & Functionality

**1. Program Finder Tool:**
- Filter by: Tuition range, format, specialization, location, GMAT requirement
- Sort by: Tuition, ranking, salary, duration
- Results: Grid or list view
- Save favorites (localStorage or user accounts)

**2. ROI Calculator:**
- Input: Current salary, expected salary, tuition, time to complete
- Output: Breakeven point, 10-year ROI, monthly payment
- Export: Email results, save PDF

**3. Comparison Tool:**
- Select 2-4 programs
- Side-by-side comparison table
- Share link
- Print-friendly version

**4. Application Deadline Tracker:**
- User can save programs
- Email reminders before deadlines
- Calendar export (iCal, Google Calendar)

**5. School Profile Pages:**
- Auto-generated from program data
- Aggregate all programs per school
- School overview + history
- Notable alumni
- Accreditation badges
