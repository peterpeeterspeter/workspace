## Technical Stack

### Platform:
- **WordPress** + Custom theme
- **Why:** Affiliate-friendly, easy CMS, plugin ecosystem

### Key Plugins:
- ThirstyAffiliates (link management)
- TablePress (program tables)
- RankMath (SEO + schema)
- Elementor/Gutenberg (page builder)
- WP Rocket (caching)

### Custom Post Types:
- `mba_programs` (program database)
- `mba_reviews` (detailed reviews)
- `mba_comparisons` (comparisons)

### Custom Fields:
- Tuition, duration, rankings
- GMAT/GRE requirements
- Salary, employment rates
- Accreditations, specializations
