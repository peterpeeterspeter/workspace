## Content Production Workflow

### For Each Page:

1. **Research Phase** (using Fury's data)
   - Pull program data from CSV
   - Verify from multiple sources
   - Gather salary, ranking, accreditation info

2. **Writing Phase**
   - Follow page template
   - Include keyword in H1, first 100 words
   - Use semantic variations naturally
   - Add LSI keywords (related terms)

3. **Optimization Phase**
   - Meta description (155 chars, keyword included)
   - Title tag (60 chars, keyword front-loaded)
   - URL structure (short, keyword-rich)
   - Header hierarchy (H1 → H2 → H3)
   - Internal links (3-5 relevant links)
   - External links (2-3 authoritative sources)

4. **Affiliate Integration**
   - Place affiliate links naturally
   - Use descriptive anchor text
   - Add affiliate disclosure
   - Track links (UTM parameters)

5. **Schema Markup**
   - Article schema (main content)
   - ItemList schema (program tables)
   - Review schema (review pages)
   - Breadcrumb schema (navigation)
   - Organization schema (homepage)

6. **Quality Check**
   - Readability (Flesch score 60+)
   - Grammar/spell check
   - Broken link check
   - Mobile preview
   - Load speed check
