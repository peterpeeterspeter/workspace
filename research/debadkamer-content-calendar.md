## üìù Production Schedule

### **Content Creation Workflow**

**Stap 1: Keyword Research (30 min)**
- Check volume, difficulty, intent
- Analyze SERP (waar ranken ze?)
- Identify content gap

**Stap 2: Outline (1 uur)**
- H1, H2s, H3s structure
- Key points per section
- Internal link targets
- External link sources

**Stap 3: Draft (2-4 uur)**
- Write content (1500-2500 woorden)
- Add AI renders (5-10 per artikel)
- Optimize for SEO (keywords in headings, alt text)

**Stap 4: Review (30 min)**
- Grammar/spell check
- Fact check (prijzen, data)
- Link check (interne, externe)
- Mobile preview

**Stap 5: Publish (30 min)**
- Upload to CMS
- Set categories, tags
- Schedule publish
- Social media share

**Tijd per artikel:** 4-6 uur

**Met 1 writer (20 uur/week):**
- 3-4 artikelen/week
- 12-16 artikelen/maand

**Target:**
- Maand 1-3: 1 writer
- Maand 4-6: 1 writer + 1 VA (support)
- Maand 7-12: 2 writers + 1 editor
